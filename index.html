Pipeline

From codebase to plan

Seven stages. One flow. Verification at every step. Every plan is a living document, not a fire-and-forget report.

I
Recon
Map the repo: stack, conventions, build commands.
II
Audit
Nine parallel subagents across all categories.
III
Vet
Advisor re-reads every cited location.
IV
Prioritize
Impact ÷ effort, weighted by confidence.
V
Plan
One file per finding, self-contained.
VI
Execute
Cheaper model, isolated worktree.
VII
Reconcile
Living documents, not fire-and-forget.
Spec

Plans that execute

Three properties that make every plan executable by a cheaper model without supervision.

⊡
Self-contained
All context inlined — exact paths, code excerpts, verified commands. No assumptions.
✓
Verification gates
Every step ends with expected output. The executor knows when it's done right.
⊟
Hard boundaries
Explicit STOP conditions — the model never improvises beyond the plan.
