S
Self-contained
All context inlined — exact paths, code excerpts, repo conventions, verified commands.
V
Verification gates
Every step ends with a command and expected output. Machine-checkable done criteria.
B
Hard boundaries
Explicit STOP conditions — so small models never improvise past their competence.
